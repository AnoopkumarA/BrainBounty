# BrainBounty Website

A modern, animated promotional website for the BrainBounty app - a fun and challenging puzzle and quiz game.

## 🚀 Features

- **Modern UI/UX**: Beautiful gradient designs with smooth animations
- **Responsive Design**: Fully responsive across all devices
- **Tech Stack**: React, Vite, Tailwind CSS
- **Interactive Elements**: Animated particles, hover effects, and smooth scrolling
- **Sections Include**:
  - Hero landing with animated elements
  - About section highlighting app benefits
  - Features showcase (Quiz & Block Fit Puzzle)
  - Game Coin System explanation
  - Privacy & Free-to-Use information
  - Responsive navigation and footer

## 🛠️ Tech Stack

- **React 18** - UI Library
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Beautiful icon library

## 📦 Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## 🎨 About BrainBounty

BrainBounty is a fun and challenging puzzle and quiz game designed to improve logical thinking, problem-solving, and math skills.

### Key Features:
- **BrainBounty Quiz**: Easy and hard math quizzes with score-based coin rewards
- **Block Fit Puzzle**: Shape-based puzzles with progressive difficulty
- **Game Coin System**: Earn coins through gameplay and unlock features
- **100% Free**: No subscriptions or hidden fees
- **Privacy-Focused**: Your data is safe and secure

## 📱 Game Modes

### BrainBounty Quiz
- Multiple difficulty levels (easy to hard)
- Skip hard questions by watching ads
- Earn coins based on your score
- Unlock themes with earned coins

### Block Fit Puzzle
- Unlock levels for 50 game coins
- Play with different shapes
- Earn coins by completing levels
- Use coins to restart or reshuffle shapes

## 🪙 Coin System

**Earn Coins By:**
- Watching rewarded ads
- Opening mystery boxes
- Playing quizzes
- Completing puzzle levels

**Use Coins For:**
- Unlocking puzzle levels
- Getting app themes
- Restarting puzzles
- Reshuffling shapes
